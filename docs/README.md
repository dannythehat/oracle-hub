Central AI command center for Footy Oracle LM babies.

- Stores deployed model metadata and performance metrics.
- Exposes JSON outputs for the Footy Oracle / Betopia app:

  - `oracle_outputs/golden_bets.json`
  - `oracle_outputs/value_bets.json`
